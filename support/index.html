FAQ

antworten auf häufig gestellte fragen

Für wen ist das Schlafsystem geeignet?

Das Schlafsystem DSFS in der Double Spring Box und im Double Spring Bed ist für alle geeignet, ob Seiten-, Bauch-, oder Rückenschläfer. Die Technik ist individuell auf Körperform und Gewicht einstellbar und sorgt so für einen gesunden Schlaf. Als Double Spring Box ersetzt die Technik den Lattenrost und bringt einen einzigartigen, ergonomischen Schlafkomfort. Zusätzlich sind die Unterfederungen individuell verstellbar. Auch im Laufe der Jahre falls sich Körperform und Gewicht verändern sollte.

Wie kann ich die Unterfederung verstellen?

Bei der Lieferung der Box sollten Sie Ihre Box oder Bett gleich auf eine ideale Position einstellen. Bei breiten Schultern (meistens also bei Männern) sollten Sie die Schulterpartie weich einstellen. So können Sie hier weiter einsinken. Also z.B. auf Position 1 (weich). Dazu die Hüfte etwas härter einstellen, z.B. auf Stufe 3 (mittelhart). Bei breiteren Hüften stellen Sie die Unterfederung im Bereich der Hüfte weicher. Sollten Sie mehr oder weniger Unterstützung in einzelnen Komfortzonen wünschen, erhöhen oder verringern sie einfach den Härtegrad durch Aus-/Ein-hängen der Zugfeder-Regulierung. Achten Sie auf eine Gleichmäßige Position beider Seiten. Zielführend ist eine gerade ausgerichtete Wirbelsäule. Schauen Sie auch gern eins unser Videos zu dem Thema Verstellbarkeit der Box.

Woraus besteht die Box?

Die Box besteht aus mehreren Zonen. Dabei werden in einen Birke-Multiplex Furnier-Rahmen verschiedene Federzonen integriert. Mehr Infos finden Sie hier. Die Materialien selbst werden auch im High End Möbelbau verwendet und stammen aus europäischer Produktion.

Wo wird die Box, Bed und Matratzen produziert?

Alle CREED Produkte entstehen in Europa. Kurze Wege und beste Qualität liegen uns am Herzen. Die Box wird in High-End-Produktionsstätten in Polen gefertigt. Das Bett wird direkt in Deutschland produziert und die Matratzen in Belgien. Spezial-Produzenten mit einzigartigem Erfahrungsschatz, auf den wir nach langer Recherche zurückgreifen konnten und je nach Bedarf in der Kapazität erweitern können.

Welche Größe(n) der Box benötige ich?

Die Boxen haben gängige Lattenrostmaße. Eine 90er Box hat die Maße 89x196cm. Eine 140er Box die Maße 139x196cm. Für ein gängiges Bett von 180cm Breite und 200cm Länge benötigen Sie zwei 90er Boxen. Wenn Ihr Bett eine Breite von 160cm aufweist, benötigen Sie zwei 80er Boxen. Bitte treffen Sie die entsprechende Auswahl bei Ihrer Bestellung.

Wie viel wiegt die Box?

Eine Box im Format 90cm x 200 cm, wiegt in etwa 42 Kilogramm. Zum Vergleich: ein einfacher Lattenrost mit massivem Holzrahmen liegt etwa bei 15 Kilogramm. Das Mehrgewicht der Box wird aber auf viele Auflageflächen verteilt. Somit ist das zusätzliche Gewicht auf den Längsleisten oder Eck-Auflage-Punkten gut aufgehoben.

Kann ich die Box einfach austauschen?

Die Box lässt sich einfach einlegen, wie ein Lattenrost. Die Auflagestellen für das Lattenrost werden also jetzt von der Box beansprucht. Diese sind ggf. in der Höhe anzupassen.

Wie wird die Box ausgeliefert?

Die Box wird über unseren Logistikpartner ausgeliefert, die mit Ihnen im Vorfeld einen Termin vereinbaren. Dafür werden Sie telefonisch kontaktiert, um einen individuellen Termin zu finden. Die Umverpackung besteht aus einer recylebaren Folie, die sie Ihrer Wertstofftonne zuführen können.

Brauche ich Werkzeug um die Box zu befestigen?

In der Regel benötigen Sie kein Werkzeug, um die Box zu befestigen. Sie legen die Box an der Stelle ein, wo sich vorher Ihr Lattenrost befand.

Was mache ich mit meinem alten Lattenrost?

Ihren alten Lattenrost können Sie dem örtlichen Sperrmüll oder einer Verwertungsstelle zuführen.

Wie viel Jahre Garantie gibt es auf die Box?

Sie erhalten 10 Jahre vollumfängliche Garantie gemäß unserer Garantiebedingungen in den allgemeinen Geschäftsbedingungen. Siehe hier.

Gibt es einen Vorteil für mich, wenn ich die Box Freunden empfehle?

Wenn Sie die Box Ihren Freunden empfehlen, gibt es je nach Aktion einen Empfehlungs-Bonus für Sie. Wir freuen uns, wenn Sie anderen eine Freude machen und uns empfehlen. Besserer Schlaf sollte kein Luxus sein, den man für sich behält.

Die Unterfederung macht Geräusche? Was kann ich tun?

Die Unterfederung von Double Spring Box und Double Spring Bed ist so ausgelegt, dass diese in Kombination mit Ihrer Matratze keine Geräusche hervorruft. Wenn es allerdings dazu kommt, dass eine punktuelle Belastung den Federweg an Ihr Maximum bringt, können die Begrenzer ein Geräusch hervorrufen. Bei normalem Gebrauch ist dies aber nicht zu vernehmen. Sollten Sie dennoch merkliche Geräusche wahrnehmen, melden Sie sich gern bei unserem Kundenservice.
